Why Poppins for Puffin

Poppins is a geometric sans‑serif typeface from Google Fonts known for its clean, friendly shapes and excellent readability from small to large sizes. Its modern forms align with our Puffin brand: approachable like the bird, yet confident and professional for an educational platform.

  • Clarity for learning: Open counters and balanced proportions aid on‑screen reading for students and educators.
  • Warmth with structure: Rounded terminals soften the geometry, striking a welcoming tone for schools.
  • International reach: Broad weight range and wide language coverage make it suitable for a global audience.
  • Consistency: One font family across headings, UI, and body ensures a unified brand voice.
  • Accessibility: Great legibility supports inclusive design and WCAG‑aligned contrast choices.

Implementation Notes for Design Engineers

🎯 Key Implementation Guidelines

  • CSS Variables: All colors and spacing use CSS custom properties for easy theming
  • Responsive Design: Components use CSS Grid and Flexbox with mobile-first approach
  • Accessibility: All interactive elements include proper ARIA attributes and keyboard navigation
  • Performance: Components use modern CSS features like backdrop-filter and color-mix for efficiency
  • Dark Mode: All components support automatic dark mode via data-theme attribute

🔧 Integration Checklist

  • ✅ Import Poppins font family
  • ✅ Set up CSS custom properties (design tokens)
  • ✅ Include core component styles from styles.css
  • ✅ Add landing page component extensions from guide.css
  • ✅ Test responsive behavior on mobile/tablet/desktop
  • ✅ Verify accessibility with screen readers
  • ✅ Test dark mode functionality

📱 Responsive Breakpoints

  • Mobile: 320px - 768px
  • Tablet: 768px - 1024px
  • Desktop: 1024px+

  <link rel="stylesheet" href="styles.css">
  <link rel="stylesheet" href="guide.css">
</head>

<!-- Theme toggle example -->
<script>
function toggleTheme() {
  const html = document.documentElement;
  const currentTheme = html.getAttribute('data-theme');
  const newTheme = currentTheme === 'light' ? 'dark' : 'light';
  html.setAttribute('data-theme', newTheme);
}
</script>
