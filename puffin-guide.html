Why Poppins for Puffin

Poppins is a geometric sans‚Äëserif typeface from Google Fonts known for its clean, friendly shapes and excellent readability from small to large sizes. Its modern forms align with our Puffin brand: approachable like the bird, yet confident and professional for an educational platform.

  • Clarity for learning: Open counters and balanced proportions aid on‚Äëscreen reading for students and educators.
  • Warmth with structure: Rounded terminals soften the geometry, striking a welcoming tone for schools.
  • International reach: Broad weight range and wide language coverage make it suitable for a global audience.
  • Consistency: One font family across headings, UI, and body ensures a unified brand voice.
  • Accessibility: Great legibility supports inclusive design and WCAG‚Äëaligned contrast choices.

Implementation Notes for Design Engineers

üéØ Key Implementation Guidelines

  • CSS Variables: All colors and spacing use CSS custom properties for easy theming
  • Responsive Design: Components use CSS Grid and Flexbox with mobile-first approach
  • Accessibility: All interactive elements include proper ARIA attributes and keyboard navigation
  • Performance: Components use modern CSS features like backdrop-filter and color-mix for efficiency
  • Dark Mode: All components support automatic dark mode via data-theme attribute

üîß Integration Checklist

  • ‚úÖ Import Poppins font family
  • ‚úÖ Set up CSS custom properties (design tokens)
  • ‚úÖ Include core component styles from styles.css
  • ‚úÖ Add landing page component extensions from guide.css
  • ‚úÖ Test responsive behavior on mobile/tablet/desktop
  • ‚úÖ Verify accessibility with screen readers
  • ‚úÖ Test dark mode functionality

üì± Responsive Breakpoints

  • Mobile: 320px - 768px
  • Tablet: 768px - 1024px
  • Desktop: 1024px+
